Tic Tac Toe Game

Using a JavaScript object to store player positions on a grid, this project links that game logic to an HTML page containing overlapping, matching play spaces, one containing an image of a tic tac toe grid drawn on a chalkboard, and one containing the CSS-drawn x's and o's.

Includes Sound!

Be sure to listen with your sound turned up! Includes chalkboard marking sounds for each player and a sarcastic-sounding celebration sound at the end.
